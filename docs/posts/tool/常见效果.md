## 滚动条颜色

```less
@primary-color: #449eff;
.changeScrollStyle() {
  &::-webkit-scrollbar {
    width: 4px;
    height: 4px;
    border-radius: 4px;
    transition: 0.3s background;
  }

  &::-webkit-scrollbar-thumb {
    background: none;
  }

  &:hover::-webkit-scrollbar-thumb {
    background: @primary-color;
  }
}
```
